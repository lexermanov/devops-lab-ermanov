Ход Работы

В ходе работы освоены базовые команды Docker для управления контейнерами и образами. Сначала изучены команды docker pull, docker run, docker ps. Поработал с интерактивным контейнером Ubuntu: зашёл внутрь, установил пакет curl и проверил его работу. Затем запущен веб-сервер nginx, открыты порты и проверена доступность в браузере. Появился опыт просмотра логов, подключения к работающему контейнеру, остановки, запуска и удаления контейнеров. Научился работать с томами (volumes): создан том, в нем сохранён файл. Удалил контейнер, создал новый и убедился, что файл не пропал.
